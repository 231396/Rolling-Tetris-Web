Rolling Tetris - Sobre o Jogo

Como jogar

Tetris é composto de um campo de jogo, em que diferentes peças chamada "tetrominós" descem do topo até o fundo da tela. Durante a descida de cada peça, o jogador pode mover as peças lateralmente, ou rotacioná-las, até que toquem uma peça que já foi colocada anteriormente. O jogador não pode impedir que as peças caiam, ou pará-las, mas ele pode acelerá-las.

O objetivo do jogo é usar as peças para preencher o máximo de linhas horizontais possíveis. Quando uma linha é preenchida, ela desaparece, e todos os blocos acima dela caem. Completar as linhas também concede pontos, e após acumular certo número de pontos, o jogador avança de fase. Além disso, a cada fase, a velocidade de queda dos blocos aumenta. O jogador pode limpar múltiplas linhas ao mesmo tempo, que também dá pontos extras.

O jogador perde quando os blocos se acumularem até o topo, e o jogador não conseguir posicionar um bloco dentro do campo de jogo por conta disso. O jogo não tem fim, o jogador pode continuar jogando e avançando fases até que sofra uma derrota.

No Rolling Tetris, existe uma peça especial (seu formato é a de um quadrado do menor tamanho). Quando o jogador preenche uma linha usando esta peça, o campo de jogo é invertido, e todas as peças começam a "cair para cima". As peças ficam depositadas no topo, e novas peças surgem do fundo da tela, e começam a ir para cima.

Detalhes Tecnicos

Tamanho do Tabuleiro

O tamanho do tabuleiro pode ser definido somente quando o jogo ainda não começou.
O valor padrão é 10x20.

Pontuação e Velocidade

Sempre que o jogador eliminar uma linha na horizontal, ele receberá uma pontuação, quanto mais linhas forem elimidadas de uma única vez, maior será a pontuação obtida.
Formula Pontos: 20 * linhas elimidadas * linhas elimidadas;

A cada 300 pontos que o jogador realizar ele passará de nível, fazendo com que a velocidade do jogo aumente.
Formula Velocidade: velocidade base - (nível * multiplicador de dificuldade)

Aleatoriedade das Peças

As peças não são escolhidas de forma completamente aleatorias. Funcionando da seguinte forma sempre que um tetrominós for selecionado, ele não poderá aparecer novamente enquanto todos os outros não aparecerem.
